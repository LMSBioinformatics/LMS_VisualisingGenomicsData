Visualising Genomics Data

The Course

This course introduces methods in R and Bioconductor to produce publication quality graphics from genomics Data.

Course material and exercises are available to view as rendered HTML slides or single page HTML at https://mrccsc.github.io/VisualisingGenomicsData/.

Setting up.

Install R.

R can be installed from the R-project website.
R 3.1.0 or higher is required for this course.

http://www.r-project.org/

Install RStudio.

RStudio can be installed from the R-project website.

http://www.rstudio.com/

Install required packages.

Having downloaded R and RStudio, some additional packages are required (rmarkdown and ggplot2).
To install these,

  • First launch RStudio
  • Install the packages in the R console 
    source("https://bioconductor.org/biocLite.R")
biocLite("GenomicRanges")
biocLite("Gviz")
biocLite("rtracklayer")
biocLite("BSgenome.Mmusculus.UCSC.mm9")
biocLite("TxDb.Mmusculus.UCSC.mm9.knownGene")
biocLite("TxDb.Mmusculus.UCSC.mm10.knownGene")
biocLite("TxDb.Mmusculus.UCSC.hg19.knownGene")

Download the material

The material can either be downloaded as a zip

wget https://github.com/mrccsc/VisualisingGenomicsData/archive/master.zip ./

or checked out from our Github repository https://github.com/mrccsc/VisualisingGenomicsData/

All material is available to download under GPL Creative Commons Attribution-NonCommercial-NoDerivatives 4.0 International License. license.

  • Plotting coverage and peak file
